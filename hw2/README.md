# Ray Tracing Practice

This project is a ray tracing practice.

With the information in ```.txt``` input file, output an ```.ppm``` file to simulate ray tracing.


## Input

* hw2_input.txt
  * Contained information
    * Camera information
    * Light position
    * Object information

## Output

* HW2.ppm

* HW2.jpg
  * Transfer ```.ppm``` file to ```.jpg```
